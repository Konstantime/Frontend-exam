Открой терминал в VS Code (Ctrl+`).

Убедись Node.js >=18: node -v. Если нет — скачай с nodejs.org.

Создай проект: npm create vue@latest my-vue-app (или npm init vue@latest my-vue-app).
​
Перейди в папку: cd my-vue-app.

Установи зависимости: npm install.

Запусти: npm run dev — откроется localhost:5173.
